Spring secure Demo
=========

Set up your local directory:
-----------------------
* git init
* git add -A
* git commit -m "Initial commit"
* git remote add origin https://<repo-url>.git
* git push -u origin master

Build project
------------
mvn clean install


URLs:
-----------
* {app_context}
* {app_context}/login
* {app_context}/login?logout
